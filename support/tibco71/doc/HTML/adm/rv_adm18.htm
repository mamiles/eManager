Default Port and Service Numbers

For convenient reference, these tables list port and service numbers with special meaning to Rendezvous components.

Rendezvous components use HTTP ports for browser administration interfaces.

Configurable daemons distributed with Rendezvous also open an ephemeral HTTPS port (to keep the daemon configuration secure against unauthorized modification). To find the actual HTTPS port that the operating system has assigned, check the daemon’s start banner or log file. The configurable daemons are rvrd, rvsd, rvsrd, rva and rvcache—but not rvd.
