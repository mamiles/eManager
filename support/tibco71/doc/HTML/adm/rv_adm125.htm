Forward RVCM Administrative Messages across Network Boundaries

Rendezvous certified message delivery software depends on administrative announcements (as well as point-to-point messages) between CM transports.

These messages travel freely within a single network segment. However, if your network consists of several segments connected by Rendezvous routing daemons, then you must instruct the routing daemons to forward the subjects in Table 13.

Routing daemons must forward these subjects in both directions—import and export.
