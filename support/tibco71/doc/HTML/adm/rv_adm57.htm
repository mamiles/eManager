Chapter 6 Secure Daemons (rvsd and rvsrd)

Release 7.0 introduces two new daemons, featuring SSL for secure connections to client program transports:

  • rvsd, the Rendezvous secure communications daemon, corresponds to rvd
  • rvsrd, the Rendezvous secure routing daemon, corresponds to rvrd

This chapter describes the security features of these two daemons, and details the parameters that differentiate them from their non-secure counterparts.
