Subject Gating

This page lets you configure subject gating (import and export subjects) for a local network.

To display this page, click the name of a local network in a row of the Local Networks Configuration page.

For background information, see Subject Gating, and Subject Filtering with Wildcards.

The local network can export subjects that match these names. You can remove a subject at any time.
adding subjects
To add subjects, specify the subject string (which may contain wildcards) here, and click one of three buttons:
  • Import
  • Export
  • Import and Export
