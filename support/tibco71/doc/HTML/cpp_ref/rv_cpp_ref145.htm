TibrvQueueGroup::add()

Method

Declaration 

virtual TibrvStatus add( 
    TibrvQueue* queue);

Purpose

Add an event queue to a queue group.

Remarks

If the queue is already in the group, adding it again has no effect.

Parameter
Description
eventQueue
Add this event queue to a queue group.
