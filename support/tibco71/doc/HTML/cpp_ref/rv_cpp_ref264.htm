Declaration 

TibrvStatus getWorkerWeight( 
    tibrv_u32& workerWeight) const;

Purpose

Extract the worker weight of a distributed queue member.

Parameter
Description
workerWeight
The program supplies a variable, and the method stores the worker weight in that variable.
