TibrvMsg::getReplySubject()

Method

Declaration 

TibrvStatus getReplySubject(const char*& replySubject);

Purpose

Extract the reply subject from a message.

Remarks

The reply subject string is part of a message’s address information—it is not part of the message itself.

If the destination subject is not set, this method passes NULL in the return parameter.

Parameter
Description
replySubject
The program supplies a variable in this parameter; the method stores the reply subject in that variable.
The method does not copy the subject string; instead it supplies a pointer into the message’s address information. The pointer remains valid as long as the message itself remains valid and unchanged in the same location. The subject pointer becomes invalid if the program destroys the message, calls TibrvMsg::setReplySubject(), or returns from the data callback method that determines the scope of an inbound message.
