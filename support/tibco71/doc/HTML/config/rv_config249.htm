RvaInformation.getState()

Method

Declaration 

int getState()

Purpose

Extract the state (running or idle) of the agent.

Remarks

This method returns one of these constants:

  • RvaProxy.IDLE_STATE
  • RvaProxy.RUNNING_STATE

This method does not interact with the component.
