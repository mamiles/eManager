RvrdProxy.getLoggingParams()

Method

Declaration 

boolean[] getLoggingParams() 
  throws ConfigurationException

Purpose

Get the flags that guide the routing daemon’s log output.

Remarks

This method returns an array of boolean flags. Use the constants (see RvrdProxy ) to index the individual flags.
