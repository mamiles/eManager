RvaProxy.removeImportSubject()

Method

Declaration

RvaProxy removeImportSubjects( 
    java.lang.String[]  subjects) 
  throws ConfigurationException

Purpose

Remove a subject for import.

Remarks

These methods stop rva from importing the subjects.

rva imports messages from clients (applets) to the network.

This method returns the RvaProxy object, so programs can conveniently chain additional method calls to the return value.

 

Parameter
Description
subject
Remove this subject from the import list.
subjects
Remove these subjects from the import list.
