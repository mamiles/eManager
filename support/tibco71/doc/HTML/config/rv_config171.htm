RvaProxy.changeState()

Method

Declaration 

RvaProxy changeState( 
    int  state) 
  throws ConfigurationException 
 
RvaProxy changeState() 
  throws ConfigurationException

Purpose

Change the state of a Rendezvous agent process.

Remarks

The first method changes to the state that the argument specifies. The second method toggles the current state.

These methods return the RvaProxy object, so programs can conveniently chain additional method calls to the return value.

 

Parameter
Description
state
Change to this state.
Supply one of two constants; either RvaProxy.IDLE_STATE or RvaProxy.RUNNING_STATE.
