class com.tibco.tibrv.config.Service 
  extends java.lang.Object

Purpose

Represent a UDP or PGM service of a Rendezvous communications daemon.

Remarks

This object represents a communications daemon’s activity on a specific network service—that is, a physical network combined with a UDP or PGM service.

The method RvdProxy.getServices() returns objects of this class.
