RvaProxy.removeSubject()

Method

Related Forms 

RvaProxy.removeSubjects()

Declaration 

RvaProxy removeSubject( 
    java.lang.String  subject) 
  throws ConfigurationException 
 
RvaProxy removeSubjects( 
    java.lang.String[]  subjects) 
  throws ConfigurationException

Purpose

Remove a subject for import and export.

Remarks

These methods stop rva from importing and exporting the subjects.

rva imports messages from clients (applets) to the network—and exports messages from the network to clients (applets).

This method returns the RvaProxy object, so programs can conveniently chain additional method calls to the return value.

 

Parameter
Description
subject
Remove this subject from the import and export lists.
subjects
Remove these subjects from the import and export lists.
