RvaProxy removeExportSubjects( 
    java.lang.String[]  subjects) 
  throws ConfigurationException

Purpose

Remove a subject for export.

Remarks

These methods stop rva from exporting the subjects.

rva exports messages from the network to clients (applets).

This method returns the RvaProxy object, so programs can conveniently chain additional method calls to the return value.

 

Parameter
Description
subject
Remove this subject from the export list.
subjects
Remove these subjects from the export list.
