Host.getSerial()

Method

Declaration 

java.lang.String getSerial()

Purpose

Get the serial number of the Rendezvous license ticket.

Remarks

Most Rendezvous components require a valid license ticket for operation. This method returns the ticket that validates the component on this host computer.
