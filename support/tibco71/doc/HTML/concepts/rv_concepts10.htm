Platform Support

Rendezvous software, release 7.1, is available on the platforms listed in this table. In general, Rendezvous supports the operating system versions listed, plus all later versions for which the operating system vendor guarantees backward compatibility (the symbol + explicitly indicates TIBCO forward support).

Rendezvous software requires multi-thread support in the operating system.
