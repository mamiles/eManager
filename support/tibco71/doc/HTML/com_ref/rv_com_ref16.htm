Tibrv

Class

Purpose

The Rendezvous environment.

Remarks

Each program creates only one instance of Tibrv, and uses its methods to open and close the Rendezvous environment, refer to Rendezvous resources, and extract version information.

All other objects require an open Tibrv environment, and are inoperative when the environment is not open. The only exceptions to this rule are the objects defined in Chapter 4, Data (namely, TibrvMsg, TibrvMsgField, and TibrvInt64).
