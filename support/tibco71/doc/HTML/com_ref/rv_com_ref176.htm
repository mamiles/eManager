TibrvCmListener.isValid()

Method

Declaration 

isValid = TibrvCmListener.isValid()

Purpose

Test validity of a CM listener object.

Remarks

Returns True if the member is valid; False if the member has been destroyed or is otherwise invalid.

Parameter
Description
isValid
Boolean
Return the validity of the CM listener object.
