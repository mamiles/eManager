TibrvSdContext

Class

Purpose

This class defines static methods for interacting with secure Rendezvous daemons.

Remarks

Each program creates only one instance of TibrvSdContext, and uses its static methods to configure user names, passwords and certificates, and to register trust in daemon certificates.

Before using any methods of this class, ensure that your programming environment reference the secure daemon TibrvsdCOM component; see Referencing the Rendezvous COM Component.

Programs that use methods of this class must also use the type library Secure Daemon TIBRVSDCOMLib (rather than TIBRVCOMLib) when defining callback methods.
