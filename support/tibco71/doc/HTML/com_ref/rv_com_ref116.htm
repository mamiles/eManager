TibrvQueueGroup.add

Method

Declaration 

TibrvQueueGroup.add queue

Purpose

Add an event queue to a queue group.

Remarks

If the queue is already in the group, adding it again has no effect.

If either the queue or the group is invalid, this method reports an error.

Parameter
Description
queue
TibrvQueue
Add this event to the queue group.
