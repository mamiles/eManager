TibrvMsg.addField

Method

Declaration 

TibrvMsg.addField field

Purpose

Add a field object to a message.

Remarks

This method copies the information into a new message field, without type conversion.

It is illegal to add a field that has both a null field name (vbNullString), and a non-zero field identifier.

Parameter
Description
field
TibrvMsgField
Add this field to the message.

See Also
