Rendezvous Error Codes

Rendezvous API error codes are 32-bit integers, with hex representations 0x80048nnn.

These constants generally correspond to error codes in the Rendezvous C API—when the COM interface constant TIBRVCOM_<name> has the value 0x80048nnn, the C constant TIBRV_<name> has the value nnn.
