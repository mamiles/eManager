TibrvCmTransport.getRelayAgent()

Method

Declaration 

relayAgent = TibrvCmTransport.getRelayAgent()

Purpose

Extract the name of the relay agent used by a certified delivery transport.

Parameter
Description
relayAgent
String
Return the relay agent name of the CM transport.

Errors

The error code TIBRVCOM_CANNOT_RETRIEVE indicates that the transport does not have a relay agent.
