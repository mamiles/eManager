TibrvQueueGroup.remove

Method

Declaration 

TibrvQueueGroup.remove queue

Purpose

Remove an event queue from a queue group.

Remarks

If the queue is not in the group, or if the group is invalid, this call reports the error TIBRVCOM_INVALID_QUEUE.

To stop automatic dispatch of a queue, remove it from the automatic dispatch queue group.

Parameter
Description
queue
TibrvQueue
Remove this queue from a queue group.
