TibrvCmTransport.isDistributed()

Method

Declaration 

isDistributed = TibrvCmTransport.isDistributed()

Purpose

Test whether a CM transport object implements a distributed queue member.

Parameter
Description
isDistributed
Boolean
Return True if the object implements a distributed queue member.
Return False if the object implements a CM transport.
