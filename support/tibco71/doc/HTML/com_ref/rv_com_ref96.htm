TibrvTimer.isValid()

Method

Declaration 

isValid = TibrvTimer.isValid()

Purpose

Test the validity of a timer.

Remarks

Notice that TibrvTimer.destroy invalidates the timer immediately, even though active callback methods may continue to run.

Parameter
Description
isValid
Boolean
Return True if the timer is valid.
Return False if the timer has been destroyed.

See Also

TibrvTimer.destroy
