TibrvQueueGroup.destroy

Method

Declaration 

TibrvQueueGroup.destroy

Purpose

Destroy an event queue group.

Remarks

This method destroys the internal mechanism of the group, leaving an unusable object. Rendezvous software automatically destroys a queue group (using this method) when program execution leaves the scope of the object, or sets the object to Nothing.

The individual queues in the group continue to exist, even though the group has been destroyed.

Programs cannot destroy the automatic dispatch queue group. Attempts to destroy it return without error.
