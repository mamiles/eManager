TibrvTransport.getNetwork()

Method

Declaration 

network = TibrvTransport.getNetwork()

Purpose

Return the network interface that this transport uses for communication.

Remarks

It is an error to call this method on the intra-process transport.

Parameter
Description
network
String
Return the network parameter of the network transport.
