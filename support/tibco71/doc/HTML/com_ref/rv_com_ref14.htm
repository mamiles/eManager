Shared Library Files

VB Programs must be able to access Rendezvous shared library files (C libraries). Table 2 details the environment variables that direct VB programs to the Rendezvous installation directory. The installation directory must contain the required shared library files.

Table 2 Environment Variables for Shared Library Files 
Platform
Environment Variable
Windows
PATH must include \<install_dir>\bin
The installation procedure sets this variable automatically.
