Multi-Threading

Table 1 compares the two models for multi-thread behavior in COM. Rendezvous 7.0 (and later) supports both models. Rendezvous objects are thread-safe in either model.

Table 1 Thread Models
Apartment Model
Free Thread Model
Single-threaded apartments (STA)
Multi-threaded apartments (MTA)
Restrictions
Programs using this model require the Windows message queue. Windows GUI programs automatically administer the Windows message queue. Use this model in the GUI thread.
Such programs can use the Rendezvous automatic dispatch queue group.
Programs must not use a GUI in threads that use this model.
Such programs can not use the Rendezvous automatic dispatch queue group; instead, they must explicitly dispatch Rendezvous events.
