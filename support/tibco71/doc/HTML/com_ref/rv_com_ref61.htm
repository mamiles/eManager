TibrvMsg.setProperty

Method

Declaration 

TibrvMsg.setProperty name, value

Purpose

Set a property on a message.

Remarks

Properties are supplementary information—they are not part of the message itself.

Parameter
Description
name
String
Set the property with this name.
value
Variant
Set this property value.

Property
Description
cmTimeLimit
Double
The CM time limit (in whole seconds) of a message. Setting this property overrides the default time limit of the sending CM transport.
Fractional values are truncated.
