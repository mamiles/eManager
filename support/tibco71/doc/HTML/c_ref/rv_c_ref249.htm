Using the Perl 5 Interface

Be sure that the library path variable is set properly; for instructions, see the README file.

Place this module reference near the top of each Perl program file that calls Rendezvous API functions: 

	use Tibrv;

If the program uses Rendezvous fault tolerance features, add this module reference: 

	use TibrvFt;

If the program uses Rendezvous certified message delivery features or distributed queues, add this module reference: 

	use TibrvCm;
