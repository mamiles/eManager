tibrvQueueGroup_Create()

Function

Declaration 

tibrv_status tibrvQueueGroup_Create( 
    tibrvQueueGroup*        eventQueueGroup);

Purpose

Create an event queue group.

Parameter
Description
eventQueueGroup
The program supplies a location, and the function stores the address of a new queue group in that location.
The queue group remains valid until the program explicitly destroys it.

See Also
