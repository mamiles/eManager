Appendix B Perl 5 Interface

Perl programs can directly call Rendezvous API functions using a Perl 5 loadable module called Tibrv, which extends the Perl language.

To simplify Perl prototyping of Rendezvous applications in C, Rv presents Perl programs with an interface that is identical to the C Rendezvous API.
