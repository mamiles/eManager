tibrvStatus_GetText()

Function

Declaration 

const char* tibrvStatus_GetText( 
    tibrv_status    status);

Purpose

Return the descriptive string corresponding to a status code.

Remarks

Status strings use the ISO 8859-1 character encoding.

Parameter
Description
status
Return the string for this status code.
