tibrvMsg_GetSendSubject()

Function

Declaration 

tibrv_status tibrvMsg_GetSendSubject( 
    tibrvMsg            message, 
    const char**        subject);

Purpose

Extract the subject from a message.

Remarks

The subject string is part of a message’s address information—it is not part of the message itself.

Programs must not modify the storage in which this subject string resides.

Parameter
Description
message
Get the subject of this message.
subject
The function stores a pointer to the subject string in this location.
The function does not copy the subject string; instead it supplies a pointer into the message’s address information. The pointer remains valid as long as the message itself remains valid and unchanged in the same location. The subject pointer becomes invalid if the program destroys the message, calls tibrvMsg_SetSendSubject(), or returns from the data callback function that determines the scope of an inbound message.
