Verify Installation

We recommend that you verify correct operation of Rendezvous software. You can verify installation at any time by invoking the command TIBRV:[COM]QUICK_TEST.COM.

The test procedure starts the Rendezvous daemon, starts a listening program, and sends three messages. The listening program receives the three messages and displays them on the screen or console.

Then the procedure prints a sample of correct output. Visually compare the sample with the actual results above it. If they match in all details except for process ID numbers and timestamps, then Rendezvous software is installed and operating correctly.
