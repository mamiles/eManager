Before You Install

Before you begin to install the software, do these items first:

  • Read Compatibility with Earlier Releases in TIBCO Rendezvous Concepts.
  • Read about any late-breaking changes, described in the package inserts and the README files. There is a README file for each platform, located on the CD-ROM, in the same directory as the installation files.
  • Read the TIBCO Software Inc. License Agreement included with the product.
  • Read the appropriate chapters for your platforms.
  • Review the section Do This First—Administrator’s Checklist in TIBCO Rendezvous Administration.
