Run the Installation Program

This procedure installs the Rendezvous software.

  1. Login as QSECOFR.
  2. On the Rendezvous distribution CD-ROM, locate the appropriate directory for your platform, as listed in Table 6. (For the latest changes, see the README file.)
    Table 6 OS/400 Distribution Directories 
    Platform
    Variant
    Directory
    OS/400 V4R3 or later
    TRDP
    <CD_device>OS400V4/AS400
  3. Type this command:
    4.       LODRUN DEV(*OPT) DIR('/OS400V4/AS40')
  4. LODRUN automatically starts RSTLICPGM, which presents a display like the one in Figure 5.

    5. By default RSTLICPGM installs Rendezvous software in a library named TIBRV.
