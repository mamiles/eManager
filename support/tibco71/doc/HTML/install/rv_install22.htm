Install the Rendezvous Product Files

The distribution CD-ROM contains the Rendezvous software for VMS platforms, as shown in Table 5.

Table 5 VMS Distribution Directories 
Platform
Variant
Directory
Alpha VMS v.7.2 or later
TRDP
<CD_device>:[VMS72.ALPHA]

  1. Locate the directory name in Table 5.
  2. Retrieve and read the release notes, which are in the sys$help directory. For example, to retrieve the release notes without installing Rendezvous software, use these commands:
    3.    $ @sys$update:vmsinstal tibrv_071 dka100:[VMS72.ALPHA] - 
   $_ options N 
 
    
The release notes are in the file sys$help:tibrv_071.release_notes.
  3. Install the product software. Specify the product name and directory on the command line to VMSINSTAL. For example:
    4.    $ @sys$update:vmsinstal tibrv_071 dka100:[VMS72.ALPHA]

The installation procedure requires your input. The most important items you supply are the target location for the Rendezvous files, and the environment to install (development or runtime environment).

The installation procedure takes less than 10 minutes.
