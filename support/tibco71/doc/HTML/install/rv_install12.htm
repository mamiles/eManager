Run the Installation Program

This procedure adds a program group for Rendezvous software.

For Windows NT only, the user ID that runs this installation procedure determines which path is updated by the setup program:
  • If you run the installation program as an administrator, it adds Rendezvous software to the machine’s path, and any user who logs in to the computer will have access to the Rendezvous directory.
  • If you run the installation program as a non-administrative user, it adds Rendezvous software to that user’s path.

  1. Start the File Manager utility or the Explorer utility.
  2. On the Rendezvous distribution CD, locate the appropriate directory for your platform and variant, as listed in Table 4. (For the latest changes, see the README file.)
