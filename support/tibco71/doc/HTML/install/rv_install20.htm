Mount the Distribution Media

Insert the Rendezvous distribution CD-ROM into the CD-ROM drive.

With VMS v.7.2 and later, use the mount command. The mount command automatically determines that the CD-ROM is formatted according to ISO 9660, but you must specify the record format for the backup save sets. For example: 

   $ mount /media_format=CDROM - 
   $_ /undefined=fix:none:8192 <CD_device> tibrv71
