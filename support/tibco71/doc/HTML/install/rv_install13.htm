Update the Execution Path

The Rendezvous SETUP program updates the execution path, but this change only becomes effective when you log in. Before attempting to use any Rendezvous components:

  • Under Windows NT and Windows 2000, log out and then log in again.
  • Under Windows 95 or 98, reboot the computer.
