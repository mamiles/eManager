User Accounts

To run Rendezvous programs, user accounts must have netmbx and tmpmbx privileges.

Rendezvous programs accept command line arguments. To set up these arguments, each user must execute TIBRV:[COM]TIBRV_SETUP.COM. As a convenience, we recommend referencing it in each user’s login.com file.
