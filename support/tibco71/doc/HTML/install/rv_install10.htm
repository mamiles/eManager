Chapter 3 Microsoft Windows

The Rendezvous installation procedures for Windows NT, Windows 2000, Windows XP, Windows 95, and Windows 98 platforms are all identical.

Windows 2000 and Windows XP support the PGM variant.

Windows 95, 98, and NT do not support PGM.
