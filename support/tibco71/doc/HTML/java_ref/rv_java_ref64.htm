TibrvIPAddr

class com.tibco.tibrv.TibrvIPAddr 
  extends java.lang.Object

Purpose

Represent an IP address.

Remarks

In general, an IP address consists of four 8-bit unsigned integers, in network byte order.

This class has no destroy() method. Instead, the Java garbage collector reclaims storage automatically.
