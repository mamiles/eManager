  throws TibrvException

Purpose

Get the value of a field as a scalar value.

Remarks

Each convenience method in this family retrieves a field and extracts its data. If the field’s type (as it exists) does not match the type of the convenience method, then the get<type> method throws an exception; in contrast, the getAs<type> method attempts to convert the data (see Decoding and Type Conversion, and Wire Format to Java Datatype Conversion Matrix). If conversion is not possible, the method throws an exception with status code TibrvStatus.CONVERSION_FAILED.
