TibrvCmMsg.getSender()

Method

Declaration 

static final java.lang.String getSender( 
    TibrvMsg    msg) 
  throws TibrvException

Purpose

Extract the correspondent name of the sender from a certified message.

Remarks

If the message is from a CM sender, then TibrvCmMsg.getSender() yields a valid CM correspondent name. If the message is not from a CM sender, then TibrvCmMsg.getSender() returns null.

Parameter
Description
msg
Extract the sender name from this message.
