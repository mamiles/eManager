TibrvCmTransport.getRelayAgent()

Method

Declaration 

java.lang.String getRelayAgent()

Purpose

Extract the name of the relay agent used by a certified delivery transport.

Errors

The error code TibrvStatus.ARG_CONFLICT can indicate that the transport does not have a relay agent.
